# 📊 Sales Revenue Analysis Dashboard (Excel)

This project contains an interactive **Sales Revenue Analysis Dashboard** built using Microsoft Excel.
The dashboard provides a comprehensive overview of sales performance, revenue distribution, and customer segmentation through dynamic visualizations and filters.
---

## 🧩 Project Overview

The dashboard analyzes sales data across multiple dimensions, including:
* Revenue by City and Year
* Revenue by Product Category
* Revenue by Order Priority
* Monthly Revenue Trends
* Customer Segment Contribution
* Product-Level Performance

The goal of this project is to transform raw sales data into meaningful business insights using Excel's analytical tools.

---

## 🛠 Tools & Features Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers (Interactive Filters)
* Data Cleaning & Transformation
* KPI Summary Metrics
* Custom Dashboard Layout & Formatting

---

## 📈 Key Insights Delivered

* Identification of top-performing product categories
* Revenue comparison across different cities and years
* Analysis of customer segments contribution
* Monthly revenue trend patterns
* Impact of order priority on revenue

## 🚀 How to Use

1. Open the Excel file.
2. Navigate to the **Dashboard** sheet.
3. Use the slicers (filters) on the left panel to explore data by:
   * Product Category
   * Order Year
   * Customer Segment
4. All charts and metrics will update dynamically.
